Установка Node (на собственный компьютер)

Задачи:

1. Установите node.
2. Выведите версию с помощью node --version.
3. Запустите по инструкции в видеоролике программу с Hello world.
4. Сделайте скриншот окна терминала с результатами работы вышеприведённых команд и поместите его в свой Git-репозиторий.
5. Прикрепите скриншот к ответу в этом форуме и URL в репозитории, где разместили скриншот.

  • Разработайте JSON-структуру для человекочитаемого представления того же документа, DTD которого разрабатывается в задании по DTD.
